03 · Mobile · GlassCard

Настоящий glassmorphism (наконец-то)

В текущей реализации GlassCard — это просто полупрозрачный контейнер без BackdropFilter. Добавляем ImageFilter.blur(sigmaX: 10, sigmaY: 10) + тонкую подсветку сверху — и карточки оживают.

06 · Mobile · Incoming Call

Приоритизируем «Открыть» — главное действие сцены

Три равноценные иконки (Сбросить · Открыть · Ответить) сейчас не подсказывают, что нажимать. У жильца в 80% случаев одна задача — открыть. Делаем её визуально доминирующей: большая primary-кнопка 80×80 по центру, остальные — меньше.
